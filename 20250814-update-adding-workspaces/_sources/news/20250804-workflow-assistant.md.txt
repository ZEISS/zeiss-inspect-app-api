# Workflow Assistant

```{eval-rst}
.. feed-entry::
```

![Workflow Assistant](../howtos/workflow_assistant/assets/inspection-basic_inspections-diameter_inspections.png)

The workflow assistant provides a new user interface for built-in ZEISS INSPECT commands or functions provided by Apps. It allows to organize these commands or functions in a user-friendly and efficient way.

See <a href="../howtos/workflow_assistant/workflow_assistant.html">How-to Guides &ndash; Workflow Assistant</a> for more information and <a href="https://github.com/ZEISS/zeiss-inspect-app-examples/tree/main/AppExamples/misc/WorkflowAssistants">App Examples &ndash; Workflow Assistants</a> for an example.
